# CLAUDE.md — olzhas-coach.kz

## Проект
Персональный сайт Олжаса Кундакбаева. Бизнес-наставник для предпринимателей и собственников бизнеса.

## Стек
- Next.js 16 (App Router, `output: 'export'`)
- TypeScript
- Tailwind CSS v4
- Framer Motion — только для аккуратных анимаций

## Деплой
Статическая сборка: `npm run build` → директория `out/`
Деплой через GitHub Actions → rsync `out/` на VPS

## Структура
```
app/           — страницы (App Router)
components/
  layout/      — Header, Footer, MobileMenu, Container
  sections/    — секции главной страницы
  ui/          — атомарные компоненты
data/          — контентные данные (НЕ хардкодить в компонентах)
public/images/ — изображения
```

## Дизайн-токены
- `--color-ink`    #0C0D0E — основной фон
- `--color-ink-2`  #17191B — карточки
- `--color-warm`   #E9E4DA — тёплые светлые секции
- `--color-bronze` #B7A27A — акцент (не ярко-жёлтый)
- `--color-text`   #FFFFFF — текст
- `--color-muted`  #A7A7A7 — второстепенный текст

## Дизайн-принципы
- Editorial luxury: дорого, спокойно, сдержанно
- Больше воздуха, меньше элементов. Mobile-first. WCAG AA.

## Запрещено
- Яркие градиенты и неон
- Анимации ради анимаций
- AI-клише: «раскройте потенциал», «трансформация», «выйдите на уровень»
- Инфобизнес: скидки, подарки, цены в открытом виде
- Неподтверждённые факты и кейсы (помечать TODO)
- Стоковые фото посторонних людей

## Правило после изменений
`npm run lint` → `npm run build` — оба должны проходить без ошибок.
